# 🔐 Настройка OAuth 2.0 приложения в Точка Банке

## 📋 Пошаговая инструкция

### 1. Вход в личный кабинет
1. Перейдите на https://enter.tochka.com
2. Войдите в личный кабинет Точка Банка
3. Перейдите в раздел "API" или "Разработчикам"

### 2. Создание OAuth приложения
1. Нажмите "Создать приложение" или "Зарегистрировать OAuth 2.0 приложение"
2. Заполните форму:
   - **Название приложения**: "Платежи для conf.ru" (или любое другое)
   - **Описание**: "Интеграция для приема платежей"
   - **Redirect URI**: `https://ваш-сайт.ru/tochka-payment/redirect/`
   - **Webhook URL**: `https://ваш-сайт.ru/tochka-payment/webhook/`

### 3. Настройка прав доступа (Scope)
**ВАЖНО!** В настройках OAuth приложения обязательно укажите следующие права:

- ✅ `accounts` - доступ к счетам
- ✅ `balances` - просмотр балансов  
- ✅ `customers` - работа с клиентами
- ✅ `statements` - получение выписок
- ✅ `sbp` - Система быстрых платежей
- ✅ `payments` - проведение платежей

**Без этих прав приложение не сможет создавать платежи!**

### 4. Получение учетных данных
После создания приложения вы получите:
- **Client ID** - публичный идентификатор
- **Client Secret** - секретный ключ (сохраните его!)

### 5. Настройка в WordPress
1. Перейдите в "Настройки → Точка Банк"
2. Введите полученные данные:
   - **Client ID**: ваш_client_id
   - **Client Secret**: ваш_client_secret
3. Сохраните настройки

### 6. Тестирование (полный OAuth 2.0 flow)
1. **Нажмите "🧪 Тестировать подключение"** - проверьте подключение к API
2. **Нажмите "🔐 Подтвердить разрешения"** - создайте consent (БЕССРОЧНО!)
3. **Скопируйте ссылку** - из поля ниже и откройте в браузере
4. **Войдите в личный кабинет Точка Банка** - подтвердите права доступа
5. **Автоматический обмен** - плагин автоматически получит финальный токен
6. **Готово!** - плагин готов к работе НАВСЕГДА!

## 🔧 Возможные проблемы

### HTTP 400 "Bad Request"
**Причина**: Неправильные учетные данные или права доступа

**Решение**:
1. Проверьте Client ID и Client Secret
2. Убедитесь, что приложение зарегистрировано
3. Проверьте права доступа в настройках приложения

### HTTP 403 "Forbidden" 
**Причина**: Недостаточно прав доступа

**Решение**:
1. В настройках приложения добавьте все необходимые scope
2. Обратитесь в поддержку Точка Банка

### HTTP 404 "Not Found"
**Причина**: Неправильный URL эндпоинта

**Решение**:
- Используйте правильный URL: `https://enter.tochka.com/connect/token`
